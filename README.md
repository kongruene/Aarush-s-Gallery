# Aarush's Cultural Immersion Week

Photo gallery of Purdue University's cultural centers — Spring 2026.

## Adding Photos

1. Put images in `public/images/<center>/` (lgbtq, bcc, aaarcc, lcc, naecc)
2. Add entries to the `PHOTOS` object in `src/app/page.jsx`
3. Push to GitHub → Vercel auto-deploys

## Run Locally

```bash
npm install
npm run dev
```

## Deploy

Connect this repo to Vercel. Every push auto-deploys.
